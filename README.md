XeLaTeX-ja package
=====================

This is an experimental package for Japanese typesetting with XeLaTeX.

## Requirements

* TeX engine: XeTeX (0.999992 or later).
* TeX format: LaTeX (2020-10-01 or later).
* Prerequisite packages:
  - expl3 (2020-12-07 or later)
  - l3keys2e
  - xparse

## Installation

```
xetex xelatexja.ins
```
